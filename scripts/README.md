# Скрипты

Каталог содержит утилиты для работы с данными телеграм-каналов: от выгрузки сообщений до анализа тональности.

## Сценарии
- `fetch_messages.py` — асинхронная загрузка сообщений из заданного телеграм-канала с использованием Telethon. Сохраняет результат в `data/messages.csv`.
- `analyze_texts.py` — частотный анализ текста сообщений, исключающий стоп-слова, и вывод 20 самых популярных слов.
- `sentiment_plot.py` — очистка текста и расчёт тональности сообщений с помощью модели `blanchefort/rubert-base-cased-sentiment`. Результаты записываются в `data/analysis_results.csv`.

## Запуск
1. Убедитесь, что установлен файл `.env` с `API_ID`, `API_HASH` и `CHANNEL` для взаимодействия с Telegram API.
2. Создайте каталог `data/`, если он отсутствует, — скрипты сохраняют результаты именно туда.
3. Выполняйте сценарии в виртуальном окружении с зависимостями из `requirements.txt`.
4. Для отображения прогресса и корректной работы анализаторов рекомендуется запускать их из командной строки, например:
   ```bash
   python scripts/fetch_messages.py
   python scripts/analyze_texts.py
   python scripts/sentiment_plot.py
   ```
