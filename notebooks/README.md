# Ноутбуки

Каталог содержит Jupyter Notebook-файлы, используемые для исследования данных телеграм-канала и проверки гипотез.

## Содержимое
- `exploratory_analysis.ipynb` — исследовательский анализ сообщений: предварительная очистка, визуализация ключевых метрик и проверка распределений.

## Рекомендации по использованию
1. Убедитесь, что зависимости из `requirements.txt` установлены, а каталог `data/` содержит файл `messages.csv`, полученный с помощью `scripts/fetch_messages.py`.
2. Запускайте ноутбук в виртуальном окружении с установленными зависимостями.
3. При необходимости сохраняйте результаты анализа в отдельные файлы или экспортируйте графики для отчётности.
